# 사용 가이드

처음 사용하시는 분을 위해 좌측 입력 항목의 의미와 전제사항을 정리했습니다.
모든 금액 단위는 **만원**이며, 모든 계산은 브라우저 안에서만 처리되고 외부로 전송·저장되지 않습니다.
각 항목 라벨 옆의 **ⓘ 아이콘**을 클릭하면 아래 표와 동일한 상세 설명을 앱 안에서 바로 확인할 수 있습니다.
기본값에서 값을 바꾼 항목에는 **↺ 되돌리기 아이콘**이 나타나 클릭 한 번으로 해당 항목만 기본값으로 되돌릴 수 있고, 한 섹션 안에 바뀐 항목이 있으면 섹션 제목 옆에 **초기화** 버튼이 나타나 섹션 전체를 한 번에 되돌릴 수 있습니다.

## 01. 기본 정보

| 항목 | 설명 |
|---|---|
| 생년(년) | 만 나이 계산, 국민연금 수령 연령 산정 등의 기준이 됩니다. |
| 인적공제 대상자 수(본인 포함) | 종합소득세 계산 시 적용되는 인적공제 인원입니다(본인 포함). |
| 시뮬레이션 시작연도 / 종료연도 | 현금흐름을 계산할 기간입니다. 시작연도가 종료연도보다 늦으면 계산되지 않습니다. |
| 물가상승률 | 공시가격 추정 등에 사용되는 연평균 상승률입니다. |
| 주택 공시가격 | 재산세·종합부동산세·건강보험 재산분 계산의 공통 기준값입니다. |
| 공시가격 기준연도 | 바로 위 주택 공시가격이 어느 연도 기준인지 지정합니다. 이후 물가상승률로 미래 공시가격을 추정합니다. 근속연수·연금개시 시점 등 다른 계산과는 무관합니다. |

## 02. DC 퇴직연금

| 항목 | 설명 |
|---|---|
| DC 원금(회사입금분) | 회사가 납입한 원금(이연퇴직소득)입니다. 근속연수·수령연차에 따라 감면되는 퇴직소득세로 과세됩니다. |
| DC 운용수익 | 원금을 초과해 발생한 운용수익입니다. 연금소득세(종합과세와 분리과세 중 유리한 쪽 자동 선택)로 과세됩니다. |
| DC 수령개시 연도/월 | 연금 수령을 시작하는 시점입니다. 연금수령한도·수령연차 계산의 기준이 됩니다. |
| DC 월 수령액(목표) | 매월 인출을 희망하는 금액입니다. 계좌 잔고가 부족해지면 자동으로 축소·소진됩니다. |
| DC 계좌 연 운용수익률 | 잔고에 매월 복리로 반영되는 예상 수익률입니다. |
| 근속연수 | 퇴직소득세 근속연수 공제 계산에 사용됩니다. 근속연수가 길수록 공제액이 커져 세금이 줄어듭니다. |

DC 원금(이연퇴직소득) 부분의 퇴직소득세 감면율은 **연금 실제 수령연차가 10년 이하면 30%, 10년을 초과하면 40%**가 적용되며, 별도 입력 없이 수령연차에 따라 자동으로 전환됩니다.

## 03. 개인연금

DC 퇴직연금과 구조가 유사합니다. 초기잔고, 수령개시 시점, 월 수령액(목표), 운용수익률을 입력하면 잔고 소진 시점까지 자동 계산됩니다.

| 항목 | 설명 |
|---|---|
| 개인연금 계좌 초기잔고 | 연금 개시 시점의 계좌 잔고 전체(세액공제 받지 않은 원금 포함 총액)입니다. |
| 위 잔고 중 세액공제 받지 않은 원금 | 연금저축·IRP 세액공제 한도(연 900만원, 2026년 기준)를 초과해 납입했거나 세액공제를 받지 않은 원금입니다. 해당 없으면 0으로 두면 됩니다. |

세액공제를 받지 않은 원금은 인출 시 **완전 비과세**이며, 항상 가장 먼저 소진됩니다. 그 이후부터 세액공제받은 원금+운용수익이 인출되어 연금소득세 대상이 됩니다.

**연금소득세 종합/분리과세 판단(1,500만원 기준)**: DC 운용수익 + 개인연금 과세대상분(한도이내분, 국민연금 제외)의 합계가 연 1,500만원 이하면 연령별 저율(55~69세 5.5%, 70~79세 4.4%, 80세 이상 3.3%)로 분리과세됩니다. 1,500만원을 단 1원이라도 초과하면 이 저율 혜택은 수령액 전체에서 사라지고, 전액을 기준으로 16.5% 분리과세와 종합과세(6.6~49.5%) 중 유리한 쪽을 선택합니다. 국민연금은 이 판단·세율 계산에서 제외되고 항상 종합과세에 합산됩니다.

## 04. 국민연금

| 항목 | 설명 |
|---|---|
| 국민연금 반영 | 체크하지 않으면 국민연금은 현금흐름에 포함되지 않습니다. |
| 정상수령 나이 월 예상액 | 국민연금공단에서 통지받은(또는 예상) 정상수령 나이 기준 월 예상액입니다. 정상수령 나이는 출생연도에 따라 만 61~65세로 자동 판정되며(아래 표), 좌측 입력 필드 아래에 현재 생년 기준 정상수령 나이가 함께 표시됩니다. 이 금액은 수령 개시 시점의 명목 금액으로 간주되어 이후 매년 물가상승률만큼 자동 인상됩니다. 국민연금공단 조회 화면에서 "오늘 기준 현재가치"로 안내받은 금액을 그대로 입력하면 실제보다 낮게 계산될 수 있습니다. |
| 조기(-)/연기(+) 연수 | 조기수령(최대 -5년, 1년당 -6%) 또는 연기수령(최대 +5년, 1년당 +7.2%)을 반영합니다. 기준점은 위 정상수령 나이입니다. |

**출생연도별 정상수령(지급개시) 나이**(국민연금법 부칙 제8조):

| 출생연도 | 정상수령 나이 |
|---|---|
| ~1952년생 | 60세 |
| 1953~1956년생 | 61세 |
| 1957~1960년생 | 62세 |
| 1961~1964년생 | 63세 |
| 1965~1968년생 | 64세 |
| 1969년생~ | 65세 |

|---|---|

## 06. 건강보험 · 임의계속가입

재취업 기간이 1년 이상이면 퇴사 후 **임의계속가입** 자격이 자동 적용되어, 직장가입자와 유사한 낮은 보험료가 일정 기간 유지됩니다(법정 최대 3년). 이 기간이 지나면 지역가입자로 자동 전환되어 재산분·소득분 보험료가 재계산됩니다.

## 07. 주택 다운사이징

| 항목 | 설명 |
|---|---|
| 다운사이징 반영 | 지정 연도에 더 낮은 공시가격의 주택으로 이전하는 것을 가정합니다. |
| 다운사이징 연도 / 후 공시가격 | 이 시점부터 낮아진 공시가격 기준으로 재산세·종부세·건강보험 재산분이 재계산됩니다. |

**주의**: 다운사이징은 재산세·종합부동산세·건강보험 재산분에만 반영됩니다. 매도 시 발생하는 **양도차익·양도소득세는 포함되지 않습니다.** 1회성 거액이라 월별 현금흐름 그래프를 왜곡시킬 수 있어 의도적으로 제외했습니다.

## 결과 화면 보는 법

- **생애주기 타임라인**: 연도별로 직장가입/임의계속/지역가입/다운사이징/국민연금 개시 등 주요 국면을 한눈에 보여줍니다.
- **KPI 카드**: 기간 총 세금·공과금, 연평균 세금, 기간 합계 세후 순수입, 순수입 최저 연도, DC·개인연금 소진 시점을 요약합니다.
- **월별 수입·지출 차트**: 수입(넓은 초록 막대)과 지출(좁은 주황 막대)이 같은 기준선(0)에서 겹쳐 그려집니다. 지출이 수입보다 큰 달에는 좁은 주황 막대가 초록 막대 위로 삐져나와 적자를 바로 알아볼 수 있습니다. 세후 순현금흐름은 파란 선으로 표시되며, 적자인 달에는 선이 0 아래로 자연스럽게 내려갑니다. 마우스를 올리면 해당 월의 수입·지출 세부 항목(연금·근로소득·세금·건강보험 등)을 확인할 수 있습니다. 연금수령한도를 초과하는 연도는 그래프 막대 영역보다 위아래로 더 크게 옅은 회색 배경으로 표시되며, 해당 연도 위에 마우스를 올리면 초과 금액·한도·적용 세율까지 툴팁에 함께 표시됩니다.
- **월별 상세 테이블**: 연도를 선택하면 해당 연도의 월별 항목별 금액(연금, 근로소득, 세금, 건강보험 등)을 확인할 수 있습니다.
- **연금수령한도 배너**: DC·개인연금이 세법상 연금수령한도를 초과하면 경고 배너가 표시되며, 초과분에는 감면 없는 세율이 적용됩니다.

## 전제사항 및 유의사항

- 본 도구는 한국 세법·건강보험료 산정방식을 근사화한 **추정치**를 제공하며, 실제 신고·부과액과 차이가 있을 수 있습니다.
- 재산세·종합부동산세는 **1세대1주택자 특례**(공정시장가액비율 45%, 종부세 11억원 공제)를 기준으로 하며, 다주택자에는 적용되지 않습니다.
- 건강보험료 재산분(지역가입자)은 공시가격 → 재산세 과세표준(45%) → 기본공제 1억원(2024.2월 개정) → 재산등급 점수표(60등급) → 점수당 금액(211.5원, 2026년 기준) 순으로 자동 계산됩니다. 1~14등급은 시행령상 확인된 구간이며, 그 이상 고액 구간은 보건복지부 공식 예시로 보정한 근사 곡선을 사용해 소폭 오차가 있을 수 있습니다. 주택담보대출 등 부채공제는 반영하지 않습니다.
- 건강보험료율은 2026년 확정치 기준입니다: 직장가입자 7.19%(본인부담 3.595%), 지역가입자 소득분 7.19%, 장기요양보험료는 건보료의 13.14%.
- 세율·공제율·건보료율은 2024~2026년 기준 세법을 반영했으며, 향후 법 개정 시 달라질 수 있습니다.
- 재산세는 7월/9월, 종합부동산세는 12월에 발생하는 것으로 가정해 월별로 배분했습니다.
- DC퇴직연금·개인연금은 계좌 잔고에서 매월 인출되며, 잔고에는 입력한 연 운용수익률이 매월 복리로 적용됩니다. 잔고가 소진되면 이후 월수령액은 0으로 자동 반영됩니다.
- 인출 재원은 비례배분이 아니라 **법정 순서**(소득세법 시행령 제40조의3)대로 소진됩니다: DC퇴직연금은 원금(이연퇴직소득)이 먼저·운용수익이 나중, 개인연금은 세액공제 받지 않은 원금(비과세)이 먼저·세액공제받은 원금+운용수익이 나중입니다. 손실이 발생하면 반대 순서(나중에 인출되는 재원부터)로 흡수됩니다.
- 예시로 채워진 기본값은 참고용일 뿐이며, 모든 금액·나이·기간·주택가격은 본인의 상황에 맞게 자유롭게 수정해 사용하도록 설계되어 있습니다.
- 정확한 세액·보험료는 세무사·국민건강보험공단·국민연금공단 상담을 통해 최종 확인하시기 바랍니다.

## 설정 저장/불러오기 및 다운로드

- **설정 저장**: 현재 입력값을 JSON 파일로 내려받습니다. **설정 불러오기**로 다시 불러올 수 있습니다.
- **엑셀 다운로드**: 연도별 요약과 월별 현금흐름 상세를 엑셀 파일로 내려받습니다.
- **리포트 인쇄**: 설정 정보, 요약, 연도별 상세, 전제조건을 포함한 인쇄용 리포트를 생성합니다.
